C. Expected value technique

Q1 --> Correct Answer B

Q1 --> Incorrect Answer

Explanation: The ‘Delphi technique’ is a commonly used tool to secure expert judgment while initiating a project. ‘Peer review’ is a project selection tool, ‘Expected value’ is a method quantitative risk analysis, and ‘WBS’ is a project planning tool.
